# Prompt Engineering Playground

A Python-based terminal application demonstrating different Prompt Engineering techniques using Google's Gemini API.

## Features

- Zero-Shot Prompting
- One-Shot Prompting
- Few-Shot Prompting
- Role Prompting
- Chain-of-Thought Prompting
- Structured Prompting
- Interview Prompting
- Coding Prompting
- Summarization Prompting
- Comparison Prompting

## Technologies Used

- Python
- Google Gemini API
- VS Code
- python-dotenv

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Prompt-Engineering-Playground.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
python main.py
```
